# getsize
Get and display the size of file or directory with binary prefixes

## Build

`$ dub build -b=release`

## Usage

`$ getsize [OPTION] [FILE]...`

```
Options:
    -h - Show this help
    -v - Show version information
    -b - Bytes
    -k - KiB
    -m - MiB
    -g - GiB
    -t - TiB
    -p - PiB
    -e - EiB
    -z - ZiB
    -y - YiB
```
